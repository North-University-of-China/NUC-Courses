# 数据库课程设计

### 说明
- 本门课程是[数据库概论](https://github.com/SkyCo/NUC-Courses/tree/master/%E6%95%B0%E6%8D%AE%E5%BA%93%E6%A6%82%E8%AE%BA)的配套课程

### 关于课程设计
- 班级里自由组队，选择SQL Server或是 MySQL，依托于[网页设计和网站建设](https://github.com/SkyCo/NUC-Courses/tree/master/%E7%BD%91%E9%A1%B5%E8%AE%BE%E8%AE%A1%E4%B8%8E%E7%BD%91%E7%AB%99%E5%BB%BA%E8%AE%BE)这门课程，完成对数据库的增删查改，需要有简单的前台HTML / CSS
- 一般选择 MySQL 进行设计
- 难度一般涉及到触发器或者是存储过程
- 推荐安装 MySQL Workbench 可视化工具
